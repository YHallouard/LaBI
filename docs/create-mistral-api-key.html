Tutoriel : Créer une clé API Mistral

Ce tutoriel vous guidera pas à pas pour créer une clé API Mistral nécessaire au fonctionnement d'Héméa.

1 Se connecter à Mistral AI

Rendez-vous sur la page de connexion de Mistral AI :

https://auth.mistral.ai/ui/login

Connectez-vous à votre compte ou créez-en un si ce n'est pas déjà fait.

2 Accéder à la section API et choisir un plan

Une fois connecté, accédez à la section API et choisissez votre plan :

  • Free : Plan gratuit avec des limitations
  • Pay as you go : Paiement à l'usage

Nous recommandons le plan Pay as you go car dans le plan gratuit, Mistral peut utiliser vos données. Une analyse de rapport de laboratoire coûte environ 2 centimes.

3 Créer une nouvelle clé API

Cliquez sur le bouton pour créer une nouvelle clé API.

Entrez un nom pour votre clé (par exemple "Héméa") et définissez une date d'expiration si nécessaire.

4 Copier la clé API

Une fois la clé créée, copiez-la immédiatement. Attention : elle ne sera affichée qu'une seule fois.

Important : Ne partagez JAMAIS votre clé API avec quelqu'un d'autre. Elle donne accès à votre compte Mistral et peut engendrer des frais.
5 Utiliser la clé dans Héméa

Lancez l'application Héméa et collez votre clé API dans le champ prévu à cet effet.

Rappel de sécurité : Votre clé API sera stockée de manière sécurisée sur votre appareil uniquement. Elle n'est jamais partagée avec des tiers et n'est utilisée que pour communiquer avec l'API Mistral pour l'analyse de vos documents.

Tutorial: Creating a Mistral API Key

This tutorial will guide you step by step to create a Mistral API key needed for Héméa to function.

1 Log in to Mistral AI

Go to the Mistral AI login page:

https://auth.mistral.ai/ui/login

Log in to your account or create one if you haven't already.

2 Access the API section and choose a plan

Once logged in, go to the API section and choose your plan:

  • Free: Free plan with limitations
  • Pay as you go: Pay per usage

We recommend the Pay as you go plan because in the free tier, Mistral may use your data. A lab report analysis costs about 2 cents.

3 Create a new API key

Click on the button to create a new API key.

Enter a name for your key (for example "Héméa") and set an expiration date if needed.

4 Copy the API key

Once the key is created, copy it immediately. Important: it will only be displayed once.

Important: NEVER share your API key with anyone else. It gives access to your Mistral account and can incur charges.
5 Use the key in Héméa

Open the Héméa app and paste your API key in the designated field.

Security reminder: Your API key will be stored securely on your device only. It is never shared with third parties and is only used to communicate with the Mistral API for analyzing your documents.
